Starter Project Instructions

  1. Pick a project from our list of projects.
  2. Sign up for a GitHub account.
  3. View our repository of project night projects.
  4. Fork the repository by clicking the button labeled "Fork". This makes a copy of the repository for you to work on.
  5. Install the GitHub app for your OS: OS X (Mac) or Windows.
  6. Launch the GitHub app.
  7. Clone your copy of the repository to your computer:
    1. Select "File > New Repository" from the menu bar.
    2. Click "Clone".
    3. Select "HackNightProjects" from the list.
    4. Click "Clone".
  8. Create a branch for your code:
    1. Select "File > New Branch" from the menu bar.
    2. Name your branch according to the name of the project you chose, lower-casing all letters and replacing spaces with underscores. For example, if your project's name were "Hazardous Waste Disposal Sites", you would name your branch "hazardous_waste_disposal_sites".
    3. Click "Create Branch".
  9. Go to "~/HackNightProjects" in your file browser of choice.
  10. Create a directory with the same name as your branch.
  11. Write a script in that directory that prints "Hello, World!" to the command line.
  12. Commit your new script to your copy of the repository:
    1. Switch back to the GitHub app.
    2. Type "Add hello world script" in the field labeled "Summary".
    3. Click the button labeled "Commit to <your branch name>".
  13. Push your script to GitHub:
    1. Select "Repository > Publish" from the menu bar.
  14. Create a pull request containing your script:
    1. Select "Repository > Create Pull Request" from the menu bar.
    2. Click the button labeled "Send Pull Request".
  15. Let your project guide know that you created a pull request, and they will merge it for you.
  16. Congratulations! You've contributed to Code for San Jose!
  17. Follow the instructions in the project issue that you selected. Every time you change your script to complete a step, repeat the following steps of these instructions:
    1. Commit your change.
    2. Push your change to GitHub.
    3. Create a pull request containing your change.
    4. Ask your project guide to merge your pull request.
